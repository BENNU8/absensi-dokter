# Aplikasi Absensi Dokter

## Deskripsi
Aplikasi Absensi Dokter adalah aplikasi berbasis web yang dibuat menggunakan bahasa pemrograman **PHP** dan **MySQL**.  
Aplikasi ini digunakan untuk mencatat dan mengelola data kehadiran dokter secara terkomputerisasi sehingga lebih rapi, efisien, dan mudah dikelola dibandingkan pencatatan manual.

Aplikasi ini dikembangkan untuk memenuhi **tugas perkuliahan** dan dijalankan pada **server lokal (localhost)**.

---

## Teknologi yang Digunakan
- PHP  
- MySQL  
- HTML  
- CSS  
- JavaScript  
- Apache Web Server (XAMPP / Laragon)

---

## Fitur Aplikasi
- Manajemen data dokter  
- Pencatatan absensi dokter  
- Penyimpanan data absensi ke database  
- Tampilan berbasis web yang sederhana dan mudah digunakan  

---

## Struktur Project
- `index.php` : Halaman utama aplikasi  
- File `.php` lainnya : Proses logika aplikasi dan pengolahan data  
- `assets/` : File pendukung seperti CSS dan JavaScript  
- Database : Menyimpan data dokter dan absensi
